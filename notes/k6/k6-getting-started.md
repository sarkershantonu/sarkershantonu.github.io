### Ubuntu Installation 
- Update ubuntu ```apt update && apt upgrade -y```
- install gpg ```apt install gpg```
- download keyring ```gpg --no-default-keyring --keyring /usr/share/keyrings/k6-archive-keyring.gpg --keyserver hkp://keyserver.ubuntu.com:80 --recv-keys C5AD17C747E3415A3642D57D77C6C491D6AC1D69```
